# Compatibility Overview

This is the ADL compatibility database: which Android devices, hardware, Linux distributions, and desktop environments work together, and how well. Every claim carries a status badge and a verification level, so you can tell tested facts from manufacturer specifications.

## How verification works

- <TestStatusBadge status="maintainer-verified" /> — tested end to end on real hardware by the maintainer
- <TestStatusBadge status="community-verified" /> — reported working by community testers
- <TestStatusBadge status="experimental" /> — works with caveats; expect rough edges
- <TestStatusBadge status="needs-testing" /> — based on manufacturer specifications; a test report is welcome
- <TestStatusBadge status="deprecated" /> — no longer recommended

Want to add a device or hardware report? See [Submit a test](/docs/compatibility/submit-a-test).
