# Welcome to Android Desktop Linux

This is the official documentation for the Android Desktop Linux (ADL) project.

ADL lets you run full Linux desktop environments on Android phones and tablets — no root access required.

## What you will find here

This documentation covers everything from initial setup to advanced configuration, organized for both beginners and experienced users.

Select a section from the sidebar to get started.
