# About Android Desktop Linux

## Project purpose

Android Desktop Linux (ADL) is an open-source documentation project that
helps users turn modern Android phones into desktop Linux computers —
**without root**. The guides walk complete beginners through installing
Termux, running Ubuntu with proot-distro, launching an XFCE desktop, and
connecting a monitor, keyboard, and mouse for a full desktop workstation.

## Contribution model

ADL is open to contributions from everyone. Changes are proposed through
issues and pull requests, reviewed by the maintainer, and merged when they
meet the project's standards — see the
[contributing guide](https://github.com/thebpandey/ADL/blob/main/CONTRIBUTING.md)
and [governance model](https://github.com/thebpandey/ADL/blob/main/GOVERNANCE.md).

**Contributors are credited when their contributions are approved** — merged
work is attributed to its author in the git history, and notable
contributions are acknowledged in the
[changelog](/docs/release-notes/changelog).
